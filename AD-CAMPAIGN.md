## 1. Campaign Structure

### Campaign: El Salto Sagrado — Spring Retreat
- **Objective:** Conversions (optimize for landing page views initially, switch to purchases once Pixel has data)
- **Daily budget:** $30/day
- **Schedule:** Start ASAP → April 9 (day before retreat)
- **Placements:** IG Stories (primary), IG Feed, IG Reels, FB Feed

### Ad Sets (3 audiences, ~$10/day each)

**Ad Set 1: Warm — Existing Audience**
- Custom audience: IG engagers (people who engaged with @solsolacestudio in last 90 days)
- Custom audience: Website visitors (once Pixel is installed)
- Custom audience: Email list (if Monica has one to upload)
- Why: Highest intent. These people already know Monica.

**Ad Set 2: Lookalike — Expand Reach**
- 1% lookalike of IG engagers
- 1% lookalike of website visitors (once Pixel has data)
- Location: Austin, TX + 50 mile radius
- Age: 28–55
- Gender: Women
- Why: Finds people similar to existing audience.

**Ad Set 3: Interest-Based — Cold Audience**
- Location: Austin, TX + 50 mile radius (include Bastrop, San Marcos, Round Rock)
- Age: 28–55
- Gender: Women
- Interests (layer 2-3 together):
  - Wellness retreat / Yoga retreat / Meditation retreat
  - Spiritual healing / Energy healing / Reiki
  - Mindfulness / Self-care / Personal development
  - Brené Brown / Glennon Doyle / Oprah's Super Soul
- Exclude: People who already purchased (custom audience from Square if possible)
- Why: Broadest reach for people who are retreat-inclined but don't know Sol Solace yet.

## 2. Budget Allocation Strategy

### Phase 1: Days 1–4 (Testing)
- Split $30/day evenly across all 3 ad sets ($10 each)
- Run 2-3 creative variations per ad set
- Goal: Identify which audience + creative combo gets best CTR and cost per link click

### Phase 2: Days 5–14 (Scale Winners)
- Kill anything with CTR < 0.8% or cost per link click > $3
- Shift budget toward winning ad set (up to $20/day for best performer)
- Duplicate winning ads into other ad sets to test
- Goal: Drive conversions at lowest CPA

### Key Metrics to Watch
| Metric | Target | Kill if below |
|--------|--------|---------------|
| CTR (link) | > 1.5% | < 0.8% |
| Cost per link click | < $1.50 | > $3.00 |
| Landing page views | Track | — |
| Purchases | Track | — |
