3 Budget Strategy

Phase 1: Days 1–4 (Minimum Launch)

  • 2 ad sets × 3 angles = 6 ads total
  • $15/day per ad set (~$5/ad/day)
  • Run Angles 1, 3, 4 (Story versions)
  • Goal: Find winning audience + angle

Phase 2: Days 5–14 (Expand)

  • Kill ads with CTR < 0.8% or CPC > $3
  • Shift budget to winning ad set (up to $20/day)
  • Add Angles 2 & 5 as fresh creative
  • Optionally add Lookalike ad set (Ad Set 3)
Kill rules: CTR < 0.8% or CPC > $3 → pause that ad/ad set

4 Assets Needed

Must-Have

  • 2–3 photos of Serana (pool, cabins, land)
  • 1 photo of Monica (leading session or candid)
  • 1 branded graphic (retreat name + dates + price, teal/gold/cream)

Nice-to-Have

  • 15-sec video/reel — property walkthrough or Monica to camera
  • Testimonials from past attendees (DM screenshots w/ permission)
  • Behind-the-scenes prep photos
Specs: Stories/Reels 9:16 (1080×1920)  |  Feed 1:1 (1080×1080) or 4:5 (1080×1350)

5 Setup Checklist

  • Install Meta Pixel on landing page (need Pixel ID — give to Douglas)
  • Create custom audiences in Ads Manager:
  • IG engagers (last 90 days)
  • Website visitors (once Pixel fires)
  • Upload email list (if available)
  • Build 1 campaign with 2 ad sets (Warm + Interest-Based)
  • Upload Angles 1, 3, 4 (Story versions) into each ad set — 6 ads total
  • Set daily budget to $30 ($15 per ad set)
  • Set schedule: now → April 9
  • Enable Advantage+ placements (prioritize IG Stories)
  • Day 4: Review performance — kill underperformers, scale winners
  • Day 5+: Add Angles 2 & 5 and Lookalike ad set if budget allows

6 Brand Voice

Tone: Grounded, intentional, warm but direct. NOT woo-woo fluffy. Think honest invitation, not sales pitch.

Key differentiator: This is NOT a spa retreat — it's ceremonial, ancestral, land-based. Use that distinction in every piece of creative.
